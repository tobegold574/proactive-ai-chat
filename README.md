## 项目介绍

Proactive AI Chat 是一个革命性的 AI 聊天应用，它不仅能被动响应你的问题，还会主动关心你，在合适的时机发起对话，就像一个真实的朋友。

## 核心特性

### 主动对话
- AI 不再只是问一句答一句
- 会在合适的时机主动发起关心和交流
- 基于时间和重要信息的智能触发机制

### 智能记忆
- 记住你的偏好、习惯和重要事项
- 自动压缩和整理重要信息
- 分层记忆系统，高效不丢失关键

### 隐私优先
- 本地存储所有数据
- 可选自定义 API Key
- 完全掌控你的隐私

### 丰富体验
- 流畅的动画交互效果
- 多种 AI 人设模板选择
- 科技蓝亮色主题设计

### 在线体验
- 无需下载安装
- 直接访问即可使用
- 支持自定义 API 配置

## 应用场景

- **日常聊天**：AI 主动关心你的生活和心情
- **学习助手**：记住你的学习进度和难点
- **工作伙伴**：提醒重要事项和截止日期
- **生活顾问**：根据你的偏好提供个性化建议

## 技术亮点

- **Next.js 16**：现代化的前端框架
- **Framer Motion**：流畅的动画效果
- **TailwindCSS**：响应式设计
- **OpenAI SDK**：强大的 AI 能力
- **Vercel 部署**：全球分发网络
